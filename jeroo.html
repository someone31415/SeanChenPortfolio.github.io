Jeroo Unit 1

Description:

In this unit, we were introduced to basic Jeroo operations and repeatedly using them. We did several labs to apply these new commands. In many of them, we had to repeatedly use certain actions.

Concepts Learned:

  • Creating new Jeroos
  • Hoping and turning
  • Creating methods
  • Planting and picking flowers
  • Repeating certain commands

Jeroo Unit 2

Description:

In this unit, we made our codes shorter and more efficient by using methods. We entered repeated actions in methods, and repeatedly used those methods to complete tasks. Also, we used the methods to organize actions, and sort certain commands so they don't get mixed up.

Concepts Learned:

  • Making codes shorter with methods
  • Repeating methods
  • Organizing methods
  • Planning out a path in a method, and applying that method further.

Jeroo Unit 3

Description:

This unit taught us about loops. The loops made the code far less repetitive, a lot shorter, and more organized. We used while loops, and in the process of using while loops, we learned about conditional statements. Many times, we had a bonus available to get back to the starting position.

Concepts Learned:

  • Creating while loops
  • Checking conditional statements
  • Making our code shorter with loops
  • Boolean operators

Jeroo Unit 4

Description:

We learned about if/else statements and using conditional statements with them. This allowed us to have our jeroos check the environment, and act based on conditional statements. That allows our jeroos to perform the same task in different situations, like the three different spiral paths. Also, we practiced with recursion, and using if statements to have methods call themselves.

Concepts Learned:

  • Using if statements
  • Checking conditional statements in if statements
  • Recursion
  • Long if statements with many else if parts
