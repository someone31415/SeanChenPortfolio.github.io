Python Unit 1: Basics

Description:

In this unit, we learned the basics of python. Our labs required us to create variables, do operations with variables and do "math tricks," and we constructed classes and use them. Also, we imported classes and used them in files that uses pygame for visuals.

Concepts Learned:

  • Creating variables
  • Doing operations with variables
  • Printing
  • Constructing Classes
  • Importing and using pygame

Python Unit 2: If statements

Description:

We went over labs and slides that contained conditional if statements. The if statements checked if certain statements were true, and performed a specific set of actions if it was. In the labs, we used runners to run a program in pygame, and used objects from imported classes. We used if statements for checking for collision, checking for game over or wining, and if objects went out of the screen.

Concepts Learned:

  • Using a runner file with imported classes
  • Using if statements to check for collision
  • Using if statements to end the game
  • Using if statements to put objects back into the screen

Python Unit 3: Loops

Description:

We went over labs and slides that taught us about loops. We used for loops and while loops to allow our game to continue, and to take in multiple inputs from the user. A lab that we did was similar to what we did in Unit 2, there was a runner and imported classes, but we included a timer, score, a coin that would move once it gets touched, and movement in all 4 directions.

Concepts Learned:

  • Using a timer
  • Keeping score and time
  • Using loops to continue the game
  • Setting random positions

Python Unit 4: Lists

Description:

We used lists in this lab to change costumes and create multiple objects from the same class file. We had a runner file where the user would control a player, whose costume changes based on the direction it is moving. Also, we had a runner where there were multiple enemies that were created in a list that needed to be avoided.

Concepts Learned:

  • Having multiple images for different costumes in a list
  • Creating classes that can create multiple objects in a list
  • Making a background that is not uniform

Final Python Project

Description:

For our final project, we needed to create a game that uses everything we learned. We were required to have variables, methods, classes, user interaction, graphics, and multiple levels. We also learned some new concepts while making the project.

Concepts Learned:

  • Using a scrolling background
  • Inserting music
  • Making a game on our own
